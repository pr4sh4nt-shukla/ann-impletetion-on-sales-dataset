# ann-impletetion-on-sales-dataset
ANN-based grocery sales forecasting pipeline for Corporación Favorita (Kaggle). Merges 7 data sources (oil prices, holidays, store metadata) across 3M+ transactions. Features deep MLP regression, one-hot encoding &amp; StandardScaler. Best RMSLE: 0.1458 (Lawn &amp; Garden). Built with TensorFlow and pandas.
